SkillsUmiejętności

When I create a page I use the following technologies: Przy tworzeniu stron wykorzystuję następujące technologie:

HTML5

It is now used language for creating web pages. It offers new opportunities, new html tags and a new sites can be more semantic (it helps search engines to correct its indexing), responsive (correctly displayed on devices with different screen resolutions, such as on mobile devices) and to provide new functionality. Jest to obecnie wykorzystywany język do tworzenia stron internetowych. Oferuje nowe możliwości, nowe tagi html dzięki czemu strony mogą być bardziej semantyczne (a to ułatwia wyszukiwarkom prawidłowe ich zaindeksowanie), responsywne (prawidłowo wyświetlane na urządzeniach o różnych rozdzielczościach ekranu, np na urządzeniach mobilnych) oraz mogą oferować nowe funkcjonalności.

CSS3

Cascading Style Sheets. This is the language used to describe the appearance of the website. CSS formatting allows you to move even the most beautiful, the most complex pages into another document - stylesheet. In this way the code is clear, simple and readable. CSS3 also offers new features not available in previous standards so your websites look more professional. Czyli Kaskadowe Arkusze Stylów. To język służący do opisu wyglądu strony www. CSS pozwala przenieść formatowanie nawet najpiękniejszych, najbardziej złożonych stron do innego dokumentu — tzw. arkusza stylów. Dzięki temu kod strony jest jasny, prosty i czytelny. CSS3 oferuje także nowe możliwości niedostępne w poprzednich standardach, dzięki czemu Twoje strony wyglądają jeszcze bardziej profesjonalnie.

JavaScript

Today's probably no (and certainly no professional) site does not work without scripts. They make it possible posting on websites self-filling the text fields, pop-up menus, slides, webmail application - and that's only a small part of the scripting capabilities. To easier to use the language and to introduce new features we have many software libraries, one of which is: Obecnie chyba żadna (a na pewno żadna profesjonalna) strona nie działa bez skryptów. Dzięki nim możliwe jest zamieszczenie na stronach samowypełniających się pól tekstowych, wyskakujących menu, slajdów, aplikacji typu webmail - a to tylko mała część możliwości skryptów. Żeby ułatwić korzystanie z tego języka i wprowadzić nowe możliwości, powstało wiele bibliotek programistycznych, jedną z nich jest:

jQuery

It is a lightweight programming library for JavaScript. It allows you to reach interesting effects animation, add the dynamic changes and perform AJAX. Most plug-ins and scripts based on jQuery working on pages without changes to the HTML code. The big advantage is also that the scripts run correctly on any browser. Jest to lekka biblioteka programistyczna dla języka JavaScript. Pozwala ona osiągnąć interesujące efekty animacji, dodać dynamiczne zmiany strony, wykonać zapytania AJAX. Większość pluginów i skryptów opartych na jQuery działa na stronach, nie wymagając zmian w kodzie HTML. Dużą zaletą jest także to, że skrypty działają poprawnie na każdej przeglądarce.

PHP

It is a programming language that generates a web page in real time. It is most often used to create a server-side scripting site. Jest to język programowania, umożliwiający generowanie strony internetowej w czasie rzeczywistym. Jest najczęściej stosowany do tworzenia skryptów po stronie serwera WWW.

MySQL

It is quite easy to use database management system. Use any data stored in the form of relationships, or tables with a fixed arrangement of columns and a variable number of rows - rows are stored in the data. Jest to dość łatwy w obsłudze system zarządzania bazami danych. Służy przechowywaniu dowolnych danych w formie tzw. relacji, czyli tabel o ustalonym układzie kolumn i zmiennej liczbie wierszy - wiersze stanowią przechowywane w bazie dane.
